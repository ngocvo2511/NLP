# Vietnamese ABSA Source Code

Thu muc nay gom data va notebook can thiet de chay lai EDA va train model.

## Cau truc

- `data/`: bo du lieu `train.jsonl`, `dev.jsonl`, `test.jsonl`.
- `notebooks/01_eda.ipynb`: notebook EDA, sinh bang thong ke va cac hinh.
- `notebooks/03_train_bilstm_crf.ipynb`: notebook train/evaluate model.
- `figures/dataset/`: noi notebook EDA luu cac hinh va file thong ke phu.

## Cach chay

1. Mo `notebooks/01_eda.ipynb` va chay tat ca cell.
2. Kiem tra cac hinh sinh ra trong `figures/dataset/`.
3. Mo `notebooks/03_train_bilstm_crf.ipynb` de train va danh gia model.
